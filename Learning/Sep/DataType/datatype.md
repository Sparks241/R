# Data Type

## 向量

### 一般向量的生成

通过`c()`函数可以完成对于向量的定义，例如本文件同一目录下的“vector.r”文件中所定义的两个二维向量，并对其进行了相加操作，输出了相加的结果。

另外也可以使用`seq()`函数来完成。

下面给出其第一种使用方法：

```r
> seq(min,max,interval)
> seq(1,9,2)
[1] 1 3 5 7 9
```

下面是第二种指定了输出的长度

```r
> seq(min,max,length.out)
> seq(0,1,length.out=3)
[1] 0.0 0.5 1.0
```

还可以选择`rep()`函数来完成，其含义为重复a参数b次。

```r
> rep(a,b)
> rep(0,5)
[1] 0 0 0 0 0
```

### 逻辑向量

本情况用于对向量中的数据进行逻辑运算，例如比较大小，下面这个例子可以适用于比较向量中每一个元素与某一个基准值的大小关系。

```r
> c(11, 12, 13) > 12
[1] FALSE FALSE  TRUE
```

我们还可以通过`which()`来筛选符合逻辑向量所代表条件的向量元素，注意函数返回值为元素在向量中的下标。

```r
> a = c(11, 12, 13)
> b = a > 12
> print(b)
[1] FALSE FALSE  TRUE
> which(b)
[1] 3
```

例如可以从一个定义好的向量（线性表）中处于某个区间的数据。

```r
> vector = c(10, 40, 78, 64, 53, 62, 69, 70)
> print(vector[which(vector >= 60 & vector < 70)])
[1] 64 62 69
```
